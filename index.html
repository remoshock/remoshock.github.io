remoshock is a computer based remote-control for shock-collars with "evil" features.

It consists of a web-based remote control, that also works on mobile phones; a randomizer program; a command line interface and a REST API.

Hardware requirements, Getting started, Installations instructions.

Receivers

Supported shock collars: PAC, Petpat, Petrainer, Wodondog, ...

Mobile friendly web remote. Command line interface. Randomizer.
